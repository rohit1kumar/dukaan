# Bot

## Technical Requirements:
- Express
- Sqlite3
- Sequelize ORM

## Installation:
- Clone the repo and move into the directory
```bash
$ git clone https://github.com/rohit1kumar/dukaan.git && cd dukaan
```
- Install dependencies and Run the server
```bash
$ npm install && npm start
```
